# 🎮 Mini Game Collection

纯 HTML/CSS/JavaScript 实现的小游戏合集，直接在浏览器中运行，无需安装任何依赖。

---

## 🐍 贪吃蛇 Snake Game

👉 **[在线试玩](https://guanhao237.github.io/game/)** | 文件：`index.html`

### ✨ 特性

- 🎮 **经典玩法** — 控制蛇吃食物，越长越快
- 🐢🐍⚡ **三种难度** — 慢速 / 中速 / 快速，开始前自由选择
- 📱 **移动端适配** — 支持触屏滑动 + 虚拟方向键
- ⚡ **逐渐加速** — 分数越高，速度越快
- 🎨 **视觉特效** — 渐变色蛇身、食物脉冲光晕、粒子特效
- 💾 **最高分记录** — 按模式独立保存到浏览器本地
- ⏯️ **暂停/继续** — 空格键随时暂停
- 🏆 **通关机制** — 占满整个网格即通关

### 🎚️ 难度模式

| 模式 | 初始速度 | 加速幅度 | 说明 |
|------|----------|----------|------|
| 🐢 慢速 | 180ms | 每50分 -4ms | 适合休闲，从容不迫 |
| 🐍 中速 | 120ms | 每50分 -2ms | 经典节奏（默认） |
| ⚡ 快速 | 65ms | 每50分 -1ms | 刺激挑战，反应考验 |

每个模式独立记录最高分。

### 🎯 操作

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → / WASD | 移动 |
| Space | 暂停 / 继续 |
| R | 重新开始 |

## ⚫⚪ 五子棋 Gomoku

👉 **[在线试玩](https://guanhao237.github.io/game/gomoku.html)** | 文件：`gomoku.html`

### ✨ 特性

- 👥 **双人对战** — 黑方先手，轮流落子
- 🖱️ **点击落子** — 点击棋盘交叉点放置棋子
- 🔍 **悬停预览** — 鼠标悬停显示落子预览
- ↩️ **悔棋功能** — 撤回上一步（双方各退一步）
- ✨ **视觉标识** — 最后一手红点标记、胜利连线高亮
- 📊 **积分记录** — 黑方/白方胜场独立统计
- 📱 **响应式设计** — 桌面端和移动端均可游玩

### 🎯 操作

| 操作 | 功能 |
|------|------|
| 点击棋盘 | 落子 |
| Ctrl+Z | 悔棋 |
| R | 重新开始 |

---

## 🚀 本地运行

直接双击对应的 HTML 文件在浏览器中打开即可。

```bash
# 或使用 HTTP 服务器
python -m http.server 8080
# 访问 http://localhost:8080（贪吃蛇）
# 访问 http://localhost:8080/gomoku.html（五子棋）
```
