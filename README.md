# 🐍 贪吃蛇 Snake Game

一个使用纯 HTML/CSS/JavaScript 实现的经典贪吃蛇游戏，直接在浏览器中运行。

## ✨ 特性

- 🎮 **经典玩法** — 控制蛇吃食物，越长越快
- 🐢🐍⚡ **三种难度** — 慢速 / 中速 / 快速，开始前自由选择
- 📱 **移动端适配** — 支持触屏滑动 + 虚拟方向键
- ⚡ **逐渐加速** — 分数越高，速度越快
- 🎨 **视觉特效** — 渐变色蛇身、食物脉冲光晕、粒子特效
- 💾 **最高分记录** — 按模式独立保存到浏览器本地
- ⏯️ **暂停/继续** — 空格键随时暂停
- 🏆 **通关机制** — 占满整个网格即通关

## 🎚️ 难度模式

| 模式 | 初始速度 | 加速幅度 | 说明 |
|------|----------|----------|------|
| 🐢 慢速 | 180ms | 每50分 +4ms | 适合休闲，从容不迫 |
| 🐍 中速 | 120ms | 每50分 +2ms | 经典节奏（默认） |
| ⚡ 快速 | 65ms | 每50分 +1ms | 刺激挑战，反应考验 |

每个模式独立记录最高分。

## 🎯 操作方式

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 方向键移动 |
| W A S D | 备用移动键 |
| Space | 暂停 / 继续 |
| R | 重新开始 |

**移动端**：点击底部方向按钮，或在屏幕上滑动。

## 🚀 本地运行

直接双击 `index.html` 在浏览器中打开即可。

或者使用任意 HTTP 服务器：

```bash
# Python 3
python -m http.server 8080

# Node.js (需要安装 http-server)
npx http-server
```

然后访问 `http://localhost:8080`。
